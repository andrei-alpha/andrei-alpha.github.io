Problem MaxQ

Will is playing a game with an array of numbers. He has an initial array V made of N integers (numbered from 0 to N-1) and can perform the following operations:

  • Change the value stored on position p with another integer
  • Finding the subsequence of maximum sum between indices a and b;

    Requirement

    Help Will to perform the operations quickly.

    Input

    Input contains on the first line number N representing the size of the array. On the next line there are N elements of the array representing the initial values. The next line contains M, representing the number of operations. On each of these M lines the M operations are described in the following form:
  • 0 i p: the value on position i takes value p
  • 1 a b: your program has to find the maximum sum subsequence between a and b (a ≤ b);

    Restrictions

  • 1 ≤ N ≤ 200 000;
  • 1 ≤ M ≤ 200 000;
  • All the numbers in the array belong to the interval [-100 000, 100,000];
  • We define a subsequence as a set of consecutive terms of vector and its sum what is obtained by adding the elements that compose it;
  • There is at least one question.
  • For 20% of the points is guaranteed N ≤ 5000

    • Output

    Output must contain a number of lines equal to the number of questions from the input. Each line is required to display a single number representing the maximum sum that can be achieved in the context of the question (i = 1,2,...); from input. Where there is a negative sum display 0.

    Sample Input

    5
1 -10 4 -1 9
4
1 0 3
0 1 1
1 0 3
1 2 4

    Sample Output

    4
6
12
